# ACM-AC-ANIM-Extension

## 概述
1. 这是一个浏览器插件（扩展），实现在算法竞赛 OJ 平台刷题 **Accepted** 时播放**自定义结算动画**  
目前只支持世界最大的算法竞赛 OJ 平台 **Codeforces**
2. 这个 idea 是2025年10~12月玩魔裁的时候诞生的，其实现工作在2026年1月末2月初完成  
3. 项目结构  
整个项目，即仓库，是 **ACM-AC-ANIM-Extension** ，其中的 **ACM-AC-ANIM** 是插件本体，  
下载到本地后可以直接由浏览器“加载解压缩的扩展”来使用  
如果读者不熟悉这个操作，请参看 **使用说明.docx**  
在仓库根目录下，也就是与 ACM-AC-ANIM 同级的位置，是 **README.md** 和 **使用说明.docx** ，它们都用于介绍这个项目，本身不起实质作用  
再度强调，“加载解压缩的扩展”，处理的是 **ACM-AC-ANIM** 文件夹，而不应是其他任何文件夹  

## 注意事项
1. **在等待评测结果时请不要刷新页面**，这可能会影响结算动画的播放
2. 视频文件（ACM-AC-ANIM下的assets下的accepted.mp4）不建议过大
3. 进入submission界面时，可能会有一次动画播放，但影响不大，暂时不作修复
4. 关于更便捷的动画设置页面以及进一步的延迟优化，暂时认为没有必要，亦且不作处理
5. 暂不支持其他 OJ 平台
6. 插件的安全性并未经受绝对可靠的长期的测试，如果担心潜在的风险，请不要使用
7. 效果演示视频较大，无法上传，后续或许会在bilibili发布，届时再行注明
8. 欢迎完善和改进
